# [핵심 원칙 (Core Rules)]

1. **무단 수정 금지 (No Unauthorized Changes)**
   - 요청된 범위 외의 코드는 절대 수정, 추가, 삭제하지 마십시오.
   - 만약 로직상 추가 변경이 필요하다고 판단되면, 코드를 건드리기 전에 반드시 사용자의 허락을 먼저 구하십시오.

2. **언어 설정 (Language)**
   - 모든 대화와 설명, 그리고 코드 내 주석(Comment)은 반드시 **한국어**로 작성하십시오.

3. 주석처리
   - 모든 컴포넌스, 함수, 변수, 데이터등 상단에는 주석처리 진행해주십시오


4. **추측 금지 (No Hallucination)**
   - 프로젝트의 의존성(Dependency)이나 폴더 구조 등이 불확실할 경우, 임의로 추측하여 코드를 작성하지 말고 반드시 사용자에게 먼저 질문하여 확인하십시오.
